新浪面经

(2016年10月21日 周五 小雨)

整体说来被虐了。。。
很久没面试了，最近投了简历，昨天收到新浪的电话邀请，去新浪总部面试。在中关村软件园，对面是网易，坐公交坐了很久。
约的10点，我大概10点20到的大厦门口，打电话说出来接我一下，然后给了份笔试题，大概记忆的题目如下：
1，加入新浪的原因，离职原因
2，1-3年的规划
3，掌握的语言，操作系统
4，java 中 强 软 弱 虚 引用的特点和用处
5，hashcode和equals什么时候需要被重写
6，activity生命周期和触发时机
7，解释性语言和编译醒语言的区别，java是哪一种？
8，android遇到的内存溢出问题？解决方案？
9，类的私有成员能不能被重写？
10，随机抽奖（1到20000）中随机出2002个
11，10000条新闻中筛选出评论量最多的100条
12，字符串出现次数计算
string[] arr= {“this”,”is”,”s”,”is”,”this”};
依次输出
“this ：2”
“is”:3

差不多笔试就这么多，接下来是面试，来了两个技术，说实在的我紧张了，本身语言组织能力就不算好，再加上紧张，更悲剧了。大概问了：

其它

1，自我介绍
2，提升是自己学习重要还会平台重要？

Java相关

1，java private protected
private成员变量能否被子类修改
default 友元成员变量 的访问权限
2,java和c++比较，为什么android用java不用c/c++
3，java特性 多态的实现
4，基础数据类型 数组属于基础数据类型么
5，基本数据类型有什么？int占几个字节？unsigned int 表示范围,char 占几个字节？在不同系统机器上会不一样么？
6，final修饰的类有哪些？有什么作用？
7，java里面堆区和栈区都存储什么？
8，linux 查看进程命令，查找一个文件，查找一个文件夹下字符串的命令？
9，spring 面向切面是怎么实现的？
10，mysql数据库 事物
11，static 修饰什么，静态内部类的用途
12，描述下什么是ajax，jquery怎么请求服务器

Android相关

1，自定义view
2，jni/ndk 的底层机制，哪个项目中用到过
3，service 怎么销毁，service怎么和activity传递消息
4，你用过哪些布局，布局是如何加载的
5，事件分发机制
6，多线程 volitale关键字的实现，java中的加锁，是锁对象还是锁类
7，范型，什么时候使用过范型？

大概问我说我还有什么要了解的么？然后就让我自己坐电梯下去了，已经12点15左右了，5层应该是，话说我等电梯后又遇到那俩哥们了，貌似他俩没看见我，听见他俩说什么基础，服务，宕机什么的，不知道是不是在说我，说没说我也感觉这次没戏了，问的东西还算基础，只能怪自己基础不够扎实吧，有一些该记忆的东西没有记牢，吸取教训吧。

QQ阅读面经

(2016年10月24日 周一 阴)

阅文集团在中关村数字山谷A座5层，进去看着工位密密麻麻的，感觉很紧的样子，面试官不错，然后主要的面试内容是结合笔试答卷展开的。

1，java基本的输出，&，&&，<<等运算符的运算
2，写出两种单例模式
3，ui线程和主线程的通信方式（多种）
4，requestLayout，invalidate,postInvalidate的区别，invalidate调用的时候会不会掉用requestLayout，requestLayout会不会调用父级的重绘？
5，开源项目用过哪些？imageloader的使用，具体用到了哪些技术实现？
6，jni c++方法怎么使用java对象
7，设置http请求超时时间和代理
8，java加锁机制，互斥锁可以修饰哪些？
9，线程的几种状态？
10，线程池怎么创建？
11，渠道分发，gradle打包
12，broadcast生命周期，怎么和activity进行通信？
13，android的长按事件是怎么实现的？如果没看过源码，自己实现。
14，eventBus的实现原理，是怎么在ui线程里处理非ui线程刷新页面的操作？
15，Integer a = new Integer(12);
Integer b = new Integer(12);
boolean r = a == b;
print(r) 是true还是false
16，常用的布局，使用场景。
